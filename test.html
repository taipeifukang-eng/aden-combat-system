⚙️ 快速開始步驟

  1. 設定 Supabase：
    • 前往 Supabase.com 建立專案
    • 執行 database_schema.sql
    • 在 config.js 填入您的 URL 和 Key
  2. 本地測試：
    • 直接開啟 index.html
    • 或使用 npm run dev 啟動開發伺服器
  3. 部署至 Vercel：
    • 執行 vercel 命令
    • 設定環境變數後部署
